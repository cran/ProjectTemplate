Contributing to ProjectTemplate

If you have changes you'd like to make to ProjectTemplate, we'd love to see them. We'd prefer that you fork ProjectTemplate through GitHub and submit a pull request. If that's not possible, please e-mail a patch to the mailing list so that we can manually incorporate your changes.
